1、检查自己本机是否安装brew，使用brew list。如果提示brew命令无法识别，则未安装brew。打开终端，使用如下命令安装brew
      ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
2、使用
      brew list
    命令查看你是否安装了bash-completion。如果回车之后控制台打印bash-completion。则已经安装了。如果没安装则使用
      brew install bash-completion
    命令安装bash-completion
3、安装之后使用
      brew info bash-completion
    命令，将在控制台打印如下内容
      ==> Caveats
      Add the following lines to your ~/.bash_profile:
      if [ -f $(brew --prefix)/etc/bash_completion ]; then
      . $(brew --prefix)/etc/bash_completion
      fi
    意思是将if...fi中的内容添加到~/.bash_profile文件中。(如果没有.bash_profile则新建一个)，添加之后，重启终端。以上为安装bash-completion部分
4、接下来将git源码clone到本地。在终端中输入如下内容（前提是本机已经安装git，如果没有请先安装git）
      git clone https://github.com/git/git.git
    clone下的源码将存放于你当前所在的目录下。并默认以git命名。将git/contri/completion路径下的git-completion.bash文件拷贝到~/下。
    并重命名为.git-completion.bash
      cp git-completion.bash ~/.git-completion.bash
5、使用
      source ~/.git-completion.bash
    使配置生效。
以上为全部安装过程。试验环境OS X EI Capitan 版本 10.11.1 
PS:在安装brew的过程中出现can't get acess 某个网站，网址具体忘记了。意思是不能连接到某个网站，我就翻墙了，先运行
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall)"
  卸载之前的安装，再运行
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  重新安装成功。
